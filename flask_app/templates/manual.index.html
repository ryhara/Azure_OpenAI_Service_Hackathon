User's Manual

This page will help you understand how to use and the features of our chat application.

Judge Fairness by Chat


Features

  • This chat can judge the fairness and error of the uploaded PDF or text.
  • As a result of the judgment, it will output "This manuscript does not contain any bias" or "This manuscript contains incorrect information or bias. Please correct it."
  • If errors are included, "Discrimination/Prejudice" or "Error" will be returned along with the corresponding sentence.
  • This chat will not respond to other sentences.
  • Users can use these responses as a reference to reconsider their manuscript.

How to Use

  1. Enter the manuscript you want checked in the PDF or text box.
  2. Please Click Send Button or Shift + Enter
  3. Wait for the response.
  4. Check the response. The user views the response and reconsiders the manuscript.
  5. Repeat the above steps until the user is satisfied with the manuscript.

Caution

This chat may not be able to completely eliminate mistakes and AI bias, but it will make you realize things that you would not notice on your own.

Feedback (implementing)


Features

  • In Feedback, enter the feedback collected from people who listened to your presentation.
  • It extracts features from a large amount of feedback and tells you.

How to Use

  1. Enter the collected feedback in the PDF or text box.
  2. Please Click Send Button or Shift + Enter
  3. Wait for the response.
  4. Check the responses and use them for your next presentation.

Caution

We cannot guarantee that all feedback can be successfully extracted.

However, it will give you a summary so you don't have to read everything.

Image Search


Features

  • Image Search labels uploaded images and stores them in storage.
  • It takes input as text and outputs an image with a label that is highly relevant to the input.
  • This feature supports selecting images when creating slides
  • By registering images that are likely to be used for slide creation in advance, it will be easier to find them when you want to use them.

How to Use

Input

  1. Select image in the input box on the left
  2. Press the upload button to save the image to storage
  3. At this time, the image name and label are also saved in the database.
  4. Follow the steps above to register multiple images.

Search

  1. Enter a sentence or a list of words in the input box on the right.
  2. Then, an image related to the entered sentence or word will be output.

Caution

Only registered images can be used. Relevance is determined by labeling images. If the labels are incorrect, the correct results may not be returned.

There is a limit to the number of images that can be registered. Please do not exceed 20

Functions such as image deletion and list will be implemented in the future.
